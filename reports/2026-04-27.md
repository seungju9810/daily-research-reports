## 3. Advisor's Critical Evaluation

### Overall Grade: C

Solid engineering output in terms of volume and methodological rigor. But the scientific picture now visible from these results is alarming, and the response to that alarm — writing a paper draft — is exactly backwards. A C reflects real work done while flagging that the work is heading in the wrong direction.

### What Was Done Well
- **Evaluation design is methodologically sound.** Global HP selection aligned to SwiFT's Optuna structure, per-fold threshold via Youden's J, fold-mean as primary metric, bootstrap CI, permutation with the same test statistic — this is properly constructed and avoids the most common leakage mistakes in neuroimaging ML papers.
- **sync_results.py is a smart workflow decision.** Auto-generating LaTeX macros from result JSONs prevents copy-paste errors and makes the paper self-consistent when results change.
- **rACC result (p=0.096) is worth flagging.** The most interesting empirical signal was detected and is now quantified. This is the only baseline that hints at real signal.
- **~10 hours of committed work.** The student worked from 01:00 through 11:25 KST, which shows sustained effort.

### Concerns & Weaknesses

- **Every single model scores at chance level, including SwiFT, and the student is writing a paper.** F0 at 0.506, F1a at 0.487 (below chance), dACC at 0.516, F3 at 0.530. SwiFT's best strategy (LoRA) is 0.543. The null hypothesis — that none of these features carry information about SSRI treatment response in this sample — cannot be rejected for any of them. Writing a Nature Medicine draft at this stage is not premature confidence; it is scientific malpractice in the making.

- **F1b dACC fails to replicate Zhukovsky et al. 2025 (JAMA NO), which reported AUC 0.62–0.67 in the same EMBARC SSRI arm.** This is a 10-15 AUC-point gap. The student has not acknowledged this discrepancy in any commit message and has not attempted to diagnose it. Possible causes include: incorrect ROI index mapping (Schaefer-200 parcellation label mismatch), different split structure, different preprocessing pipeline, or feature construction error. This must be investigated before any baseline table is accepted as valid.

- **F1a falls below chance (0.487).** A properly regularized linear SVM on a binary classification task should reach at least 0.50 by construction unless there is a data issue. This is a red flag for a bug (e.g., label leakage in the wrong direction, incorrect fold assignment, sign flip in predictions) that has not been investigated.

- **7 of 11 commits co-authored with Claude.** This includes the entire paper rewrite from Korean to English. Writing is the core intellectual output of a PhD — the reasoning, argumentation, and synthesis of evidence into a coherent scientific narrative. Outsourcing this to an LLM means the student is not developing the most important skill required to become a world-class researcher. Generating methods text via AI when the underlying results are unreliable compounds the problem: the paper now has polished prose describing results that may not survive scrutiny.
